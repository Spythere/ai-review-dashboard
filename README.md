# AI Review Dashboard
AI Conversation Review Dashboard mock page made with React, TS and Material UI.

## Prerequisites

- Node.js (preferred: v24)
- Package manager of choice - preferred: yarn

## How to run

1. Run `yarn` to install dependencies
2. Run `yarn build` to build the project
3. Open project as a preview: `yarn preview` or in development mode: `yarn dev`

Live demo is available here: https://ai-board.spythere.eu/
